Collect some data

ControlShark has a robust data collection system that is ideal for deployment in the field or factory. We know data collection is not always seen as a value adding activity on the ground. That's why we have made it fast and user friendly, as well as providing "good / not good" feedback to add value for any operatives using it.

Open Data Entry Screen#

Navigate to the process screen that we just created and click on the blue button at the top to open the data entry screen.

Add new observation#

The add new observation screen will open, which will ask for the subgroup unique ID and the phase of the process it is related to. In our example we can use the subgroup ID "1907" and the phase will automatically default to the latest entry which is "Baseline".

The next screen allows you to record the measurements for this observation. Because we have chosen a subgroup of 5, we have the option to capture 5 measurements.

Speed Tip

For speed of use you can enter the measurement and simply press enter to quickly progress to the next one.

Good / Not Good Gauge

Notice that the data entry form also acts as a good / not good gauge for the data being entered which can be beneficial for operator facing quality control applications.
