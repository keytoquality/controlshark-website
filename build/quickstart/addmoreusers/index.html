Add More Users

Create individual user accounts within the organisation#

The organisation administrator can setup individual user accounts for other people in the organisation who will use ControlShark.

  1. Login to ControlShark with the administrator account
  2. Click on Manage Organisation in the top right hand corner of the screen

  1. In the Add New User section, enter the users email address into the box and click Add User

  1. An email invitation will be sent to the user for them to log in and start using ControlShark

  2. Any additional users will need to create a password before logging in for the first time

Assign permission groups to additional users#

ControlShark allows an organisation's users to have certain permissions based on their role. Simply go to Manage Organisation and click on Edit User Permissions
