Create a Process Group

Navigate to the organisation dashboard and click on the "add new process group" button.

On the next page that appears, type a name and description for the new process group. For this example we will use quality control measurements related to an orange juice product.

  • Type "Orange Juice Quality Control" into the Name field
  • You can also enter a description if you like

any field marked with an asterix (*) denotes a required field

Once finished, click the save button and you will be returned to the organisation dashboard to see your newly created process group.

Nice! Now the boring part is out of the way, let's add a process and start doing some analysis...
